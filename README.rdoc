== README


* Ruby version

    ruby 2.2.2p95

* System dependencies

    postgres (PostgreSQL) 9.4.4


* Configuration

    bundle install
    set RAILS_ENV = production


* Database creation

    rake db:create
    rake db:migrate

    Go into the seeds.rb file to configure defaults, then
    rake db:seed



Please feel free to use a different markup language if you do not plan to run
<tt>rake doc:app</tt>.
